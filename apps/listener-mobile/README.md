# App oyente móvil (Expo)

Cliente **oyente** para iOS y Android — escuchar música, biblioteca, búsqueda.

Equivalente móvil de `listener-web`. Inspiración: app principal de Spotify/Tidal.

## Fase 3B

- Home (destacados), búsqueda, reproductor, biblioteca
- Login/registro `USER`
- Streaming vía URLs firmadas de la API
- Background playback

## Arranque (cuando esté implementado)

```bash
pnpm listener:mobile:dev
```
