# Backend — Spring Boot API

API REST `/api/v1` para autenticación, catálogo, streaming y regalías.

## Estado

Pendiente — **Fase 1**.

## Módulos planificados

- `auth` — JWT, refresh tokens, RBAC
- `catalog` — artistas, álbumes, canciones
- `playback` — registro de reproducciones ponderadas
- `storage` — subida y URLs pre-firmadas (MinIO/S3)
- `royalties` — cálculo mensual (Fase 4)
